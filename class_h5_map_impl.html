std::optional< h5gt::Group > H5MapImpl::addAttributeMap ( H5Map * map,
std::string name = "" )
overridevirtual

Add attribute map.

Check if current map and attribute map reside in the same hdf5 container or not (create 'Internal' or 'External' attribute map).
The nX and nY must be equal.
Attribute map is a regular H5Map but resides within parent map.

Parameters
mapmap to be used as attribute
namename for an attribute map
Returns
Group of newly created attribute map

Implements H5Map.
